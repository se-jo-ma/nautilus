## Learnings
- HIPAA rule files follow same YAML structure as NIST rules (template, conditions, effects)
- phi_source_tag template provides source_id, phi_level, purpose, covered_entity slots
- Salience bands: 170-190 (deny), 130-150 (scope_constraint), 110-120 (escalate)
- PHI hierarchy: de-identified < limited < standard < sensitive
