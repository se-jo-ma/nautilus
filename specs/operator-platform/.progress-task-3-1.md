## Learnings
- AuditReader seek-based pagination can skip up to ~3 lines at page boundaries due to readline() after mid-line seek; tests accommodate this with >= threshold instead of exact 200 count
- JSONL test fixtures must wrap AuditEntry inside AuditRecord with nautilus_audit_entry metadata key for double-parse to work
- The except clause in audit_reader.py uses comma syntax (ValueError, Exception) which is Python 2-style but works as a catch-all in Python 3
