## Learnings
- DenialRecord requires `rule_name` field (not `rule_id`)
- Admin routes use proxy_trust auth mode for simpler test setup (just set X-Forwarded-User header)
- AuditReader can be patched at `nautilus.ui.router.AuditReader` for decision route tests
- Router returns HTMX partials when HX-Request: true header is present; full pages otherwise
